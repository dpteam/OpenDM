OpenGM
======

## Free and Open-Source of the SA:MP gamemode

### Current Working on:
 * Weapon Giving
 * Animations Playing
 
### Plans on Future:
 * All Game-Modes compilation
 
### How to Install:
 * 1. Download latest SA:MP server <sa-mp.com>
 * 2. Download any of each gamemode
 * 3. Compile gamemode with pawno
 * 4. Edit the server.cfg
 * 5. Run server.
